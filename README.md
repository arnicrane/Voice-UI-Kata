# Customer Voice Hotel UI + API Server

## Requirements

* Git
* Nodejs v10+ & NPM v5.51+ (`brew install node` on a MAC)

## Running Locally

### Backend API (incl. basic JSON DB)

- Open a terminal and navigate to this folder. Then:

    ```bash
    cd backend-api
    npm install
    npm run server
    ```
